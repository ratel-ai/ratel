# `@ratel-ai/ai-sdk-adapter`

The [Vercel AI SDK](https://sdk.vercel.ai) adapter for [Ratel](https://github.com/ratel-ai/ratel): `ratel(config).adaptTo(aiSdk())` gives the framework-neutral core `ai@7` native tool and message shapes, so an AI SDK agent discovers and invokes tools through Ratel's capability funnel with per-turn recall.

Documentation lands with the implementation.
