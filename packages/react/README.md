# @agentified/react

React hooks and components for Agentified.

## Installation

```bash
npm install @agentified/react @agentified/sdk react
# or
pnpm add @agentified/react @agentified/sdk react
```

## Usage

```typescript
import { REACT_VERSION } from "@agentified/react";

console.log(REACT_VERSION);
```
